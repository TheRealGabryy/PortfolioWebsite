# Gabriele Bolognese

## Installation

This project is a static HTML website and does not require any installation. Simply open the `index.php` file in a web browser to view the website.

## Usage

The website showcases the work and expertise of Gabriele Bolognese, a software engineer and AI/ML specialist. The main sections include:

- Hero section with a logo, title, and subtitle
- About Me section with a brief introduction and background

Users can interact with the website by clicking the "View Portfolio" button, which currently does not lead to any additional content.
